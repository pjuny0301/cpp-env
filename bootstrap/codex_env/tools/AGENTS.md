# Mirrored Tool Source Rules

- Start with each tool's `tool.json`.
- Read each tool's `README.md` before opening the Python source.
- Open `cpp_new.py` or `plan_doc.py` only when changing behavior or debugging bootstrap sync issues.
- Treat these folders as repo mirrors of the durable tool copies in `C:\codex-local\tools`.
- Keep mirrored source aligned with the durable tool source after edits.
