# Requirement 9

## 요구사항 원문

9\. multiselect 유형은 정답을 여러번 선택해야하는 유형, 빈칸에 차례대로 표시되게 함. 속도감있게 하기 위해서 선지에 눌리는 애니메이션은 뺌

## 구현 방식

multiselect는 선택 순서대로 상단 슬롯을 채우는 `multiselect_session_state`를 둔다. 선지 눌림 scale animation은 theme가 별도로 켜지 않는 한 기본 off로 두고, 제출 시 domain의 correct option set과 비교한다.

- `codex/quiz/big_plan.md`
- `codex/quiz/big_plan.md`

## 대안

React UI에서만 처리하지 말고 domain/FSM 규칙으로 먼저 고정한다.

## 장단점

장점: 포팅과 검증이 쉬워진다. 단점: 초기에 문서와 계약을 먼저 맞추는 비용이 든다.

## 검증

domain unit test와 주요 UI flow smoke test로 검증한다.
