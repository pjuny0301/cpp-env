# Requirement 32

## 요구사항 원문

32\. 유저가 옵션에서 허용시 한 번 틀리면 그 문제가 오답노트로 이동하는 기능도 추가 유저가 설정한 숫자만큼 맞추면 본래 그룹으로 다시 이동

## 구현 방식

오답노트는 `wrong_note` learning bucket으로 추가한다. 옵션이 켜져 있으면 첫 오답 또는 N회 오답 시 이동하고, 사용자가 정한 연속 정답 수를 만족하면 본래 그룹으로 복귀한다.

## 관련 경로

- `apps/quiz/android-quiz-app`, `apps/quiz/quiz-vulkan`
- `codex/quiz/big_plan.md`
- `codex/quiz/big_plan.md`

## 대안

React UI에서만 처리하지 말고 domain/FSM 규칙으로 먼저 고정한다.

## 장단점

장점: 포팅과 검증이 쉬워진다. 단점: 초기에 문서와 계약을 먼저 맞추는 비용이 든다.

## 검증

domain unit test와 주요 UI flow smoke test로 검증한다.
