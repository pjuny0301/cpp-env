# Requirement 8

## 요구사항 원문

8\. select 유형에서 OX표시 제거. multiselect multiblank blank 유형에선 OX 표시 이전으로 되돌림

## 구현 방식

select 유형의 큰 OX 오버레이는 금지한다. blank, multi_blank, multiselect에는 유형별로 낮은 불투명도의 inline feedback을 허용하되, 화면 중앙 오버레이는 쓰지 않는다.

- `codex/quiz/big_plan.md`
- `codex/quiz/big_plan.md`

## 대안

React UI에서만 처리하지 말고 domain/FSM 규칙으로 먼저 고정한다.

## 장단점

장점: 포팅과 검증이 쉬워진다. 단점: 초기에 문서와 계약을 먼저 맞추는 비용이 든다.

## 검증

domain unit test와 주요 UI flow smoke test로 검증한다.
