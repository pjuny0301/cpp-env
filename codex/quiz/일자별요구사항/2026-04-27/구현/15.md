# Requirement 15

## 요구사항 원문

15\. 텍스트로 추출하고 텍스트문서에 파일경로가 적힌 텍스트 사진과 인덱싱하는 프로그램 개발

## 구현 방식

`content-index` CLI를 만든다. 출력은 `pdf-index/<book>/<chapter>/pages.json`, `images.json`, `paragraphs.json`이고 각 텍스트 블록은 파일 경로, page, bbox, image refs를 가진다.

## 관련 경로

- `apps/quiz/quiz-editor`, `apps/quiz/quiz-vulkan`, future worker CLI
- `codex/quiz/big_plan.md`
- `codex/quiz/big_plan.md`

## 대안

앱 내부 직접 구현 대신 외부 워커/캐시 파일 중심으로 시작한다.

## 장단점

장점: 포팅과 검증이 쉬워진다. 단점: 초기에 문서와 계약을 먼저 맞추는 비용이 든다.

## 검증

샘플 PDF 입력, 캐시 재사용, 생성 JSON schema validation으로 검증한다.
