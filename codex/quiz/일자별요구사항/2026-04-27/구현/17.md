# Requirement 17

## 요구사항 원문

17\. **수학 수식 필기 인식 (OCR):**  11번(수학 증명) 기능과 연동하여, 태블릿에서 직접 쓴 수식을 텍스트로 변환해 AI에게 전달합니다.

## 구현 방식

수학 수식 필기 OCR은 18번과 같은 기능으로 묶는다. 태블릿 stroke 또는 이미지 입력을 `math_ocr_request`로 저장하고, OCR 결과는 원본 좌표와 LaTeX 후보를 함께 반환한다.

## 관련 경로

- `build/external/tools`, `apps/quiz/quiz-editor`, `apps/quiz/quiz-vulkan`
- `codex/quiz/big_plan.md`
- `codex/quiz/big_plan.md`

## 대안

앱 내부 직접 구현 대신 외부 워커/캐시 파일 중심으로 시작한다.

## 장단점

장점: 포팅과 검증이 쉬워진다. 단점: 초기에 문서와 계약을 먼저 맞추는 비용이 든다.

## 검증

고정 샘플 입력과 expected grader/OCR output fixture로 검증한다.
