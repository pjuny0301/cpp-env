# Requirement 19

## 요구사항 원문

19\. **포커스 모드 (Zettelkasten 연동):**  퀴즈를 풀다 막히면 해당 개념과 연결된 자신의 과거 메모(옵시디언 등)를 작은 팝업으로 즉시 띄워주는 기능입니다.(AI가 자신의 손필기내용을 바탕으로 생성해서 띄워줌)

## 구현 방식

포커스 모드는 Obsidian/Zettelkasten vault indexer가 필요하다. 로컬 markdown을 chunk, concept, embedding으로 인덱싱하고, 퀴즈에서 막힌 concept id를 기준으로 작은 popup scene을 띄운다. 손필기 기반 생성물도 같은 artifact ref로 캐시한다.

## 관련 경로

- `apps/quiz/quiz-editor`, `apps/quiz/quiz-vulkan`, AI worker/cache
- `codex/quiz/big_plan.md`
- `codex/quiz/big_plan.md`

## 대안

앱 내부 직접 구현 대신 외부 워커/캐시 파일 중심으로 시작한다.

## 장단점

장점: 포팅과 검증이 쉬워진다. 단점: 초기에 문서와 계약을 먼저 맞추는 비용이 든다.

## 검증

domain unit test와 주요 UI flow smoke test로 검증한다.
