# Requirement 36

## 요구사항 원문

36\. 고급옵션에 카메라로 쓸지 태블릿으로 쓸지 선택하는 창 뜨게 할 수 있음

## 구현 방식

카메라/태블릿 선택창은 고급옵션의 proof input source selector로 구현한다. 선택값은 deck별 override와 global default를 모두 지원한다.

## 관련 경로

- `build/external/tools`, `apps/quiz/quiz-editor`, `apps/quiz/quiz-vulkan`
- `codex/quiz/big_plan.md`
- `codex/quiz/big_plan.md`

## 대안

한 번에 전체 구현하지 말고 schema, state, UI, test 순서로 나눈다.

## 장단점

장점: 포팅과 검증이 쉬워진다. 단점: 초기에 문서와 계약을 먼저 맞추는 비용이 든다.

## 검증

domain unit test와 주요 UI flow smoke test로 검증한다.
