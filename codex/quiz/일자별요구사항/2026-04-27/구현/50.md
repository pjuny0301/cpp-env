# Requirement 50

## 요구사항 원문

50\. 문제 전환하는 애니메이션 없앤다

## 구현 방식

다른 브랜치 작업으로 분리한다. 문제 전환 animation은 theme token `transition.question.enabled=false`로 끌 수 있게 만들어 merge 충돌을 줄인다.

## 관련 경로

- `apps/quiz/quiz-vulkan`
- `codex/quiz/big_plan.md`
- `codex/quiz/big_plan.md`

## 대안

렌더러 직접 분기 대신 theme/style token과 scene contract로 우회한다.

## 장단점

장점: 포팅과 검증이 쉬워진다. 단점: 초기에 문서와 계약을 먼저 맞추는 비용이 든다.

## 검증

scene snapshot, layout test, renderer smoke test로 검증한다.
