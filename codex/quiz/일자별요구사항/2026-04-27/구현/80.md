# Requirement 80

## 요구사항 원문

80\. 퀴즈앱 전반을 카툰렌더링게임처렁 만들고싶음. 아까 코덱스갸 나와서 설명하는 것도 카툰렌더링게임의 npc가 나와서 설명하는 느낌

## 구현 방식

전체 앱을 카툰 렌더링 게임처럼 보이게 하려면 2D cartoon UI theme부터 만들고, 이후 Vulkan에서 cel-shaded companion/scene layer를 추가한다. 학습 진행은 NPC dialogue와 highlight script가 이끈다.

## 관련 경로

- `apps/quiz/quiz-vulkan`
- `codex/quiz/big_plan.md`
- `codex/quiz/big_plan.md`

## 대안

렌더러 직접 분기 대신 theme/style token과 scene contract로 우회한다.

## 장단점

장점: 포팅과 검증이 쉬워진다. 단점: 초기에 문서와 계약을 먼저 맞추는 비용이 든다.

## 검증

scene snapshot, layout test, renderer smoke test로 검증한다.
